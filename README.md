 -  body-parser - Este é um middleware node.js para lidar com dados de
formulário codificados em JSON, Raw, Texto e URL.
 - cookie-parser - Analisa o cabeçalho do cookie e preenche req.cookies
com um objeto codificado pelos nomes dos cookies.

$ npm install body-parser --save
$ npm install cookie-parser --save
